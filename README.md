# Elm Starter

Boilerplate repo to get Elm project up and running in no time.

## Installation

If you haven't already

```bash
npm i -g elm elm-live
```

```bash
elm-package install
```

(Answer `y` when prompted.)


## Building

```bash
elm-live src/Main.elm --open --dir=public --pushstate --debug --output=public/elm.js
```
